Hello!

From this page you can download the extensions for Firefox and Chrome, using the above links.

  • For Chrome/Opera, right-click on the link to save the .crx file (you can't install directly from this site). Then open an extensions window and drag-and-drop the file there. ouallah
  • For Firefox, clicking the download link should prompt Firefox to directly install it.

The source can be found in the github link below. The source has instructions for building the extensions yourself if you would prefer.

Once the extension is installed it will replace the placeholder image in anon posts with the actual images they are pointing to.

For issues/feature requests, contact t3r3zi on dreamwidth.
